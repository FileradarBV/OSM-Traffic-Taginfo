## OpenStreetMap Traffic Network

Fileradar uses OpenStreetMap to construct traffic networks for analyses and simulations of road traffic.

This repository contains an overview of OpenStreetMap tags that are supported by our parser.
It can be viewed in [Taginfo](https://taginfo.openstreetmap.org/projects/fileradar_traffic#tags).

### Looking for a traffic network?

We are specialised in working with traffic networks, and can use OpenStreetMap to provide [traffic networks in any format](https://fileradar.nl/producten/wegennetwerk-nwb-of-osm/).
